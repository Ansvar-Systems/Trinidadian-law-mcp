# Belize Law MCP

## Quick Reference

- **Jurisdiction**: Belize (BZ)
- **Source**: agm.gov.bz
- **Language**: English
- **Tools**: 8 core (non-EU jurisdiction)
- **Template**: Dominican-law-mcp

## Development

```bash
npm run census     # Enumerate laws from agm.gov.bz
npm run ingest     # Download full text
npm run build:db   # Build SQLite DB
npm run build      # Compile TS
npm test           # Run tests
```
